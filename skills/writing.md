# Skill: Writing

## Purpose

Create written content such as drafts, articles, notes, or documents.

## When to Use

- Task requests writing new content
- Task mentions "write", "draft", "create", "compose"
- Output is a text document

## Instructions

1. **Understand the request**
   - What type of content?
   - What tone/style?
   - What length?
   - Who is the audience?

2. **Gather context**
   - Read any reference materials mentioned
   - Note specific requirements

3. **Create outline**
   - Structure the content logically
   - Include all required sections

4. **Write draft**
   - Follow the outline
   - Match requested tone
   - Stay within length guidelines

5. **Review**
   - Check for completeness
   - Verify requirements met

## Output Format

Markdown document with:
- Clear title
- Organized sections
- Proper formatting

## Example

**Task**: "Write a project update for the team"

**Output**: A markdown file with sections for Progress, Challenges, Next Steps
