# Skill: Organization

## Purpose

Organize, restructure, and manage files within the vault.

## When to Use

- Task involves organizing or restructuring content
- Task mentions "organize", "sort", "categorize", "restructure"
- Need to move files or create folder structure

## Instructions

1. **Assess current state**
   - List files to be organized
   - Understand current structure
   - Identify the problem

2. **Define target structure**
   - Determine logical groupings
   - Plan folder structure if needed
   - Consider naming conventions

3. **Create plan**
   - List each file and its destination
   - Note any renaming needed
   - Identify dependencies

4. **Execute organization**
   - Create new files/folders as needed
   - Move content appropriately
   - Update any cross-references

5. **Verify**
   - Check all files are in place
   - Ensure no content lost
   - Test any links

## Output Format

```markdown
# Organization Report: [Task Name]

**Date**: [date]
**Files affected**: [count]

## Changes Made

| Original | Action | New Location |
|----------|--------|--------------|
| file1.md | Moved | /folder/file1.md |
| file2.md | Renamed | file2-new.md |

## New Structure

```
/folder/
  - file1.md
  - file2.md
/other/
  - file3.md
```

## Notes

[Any relevant notes about the reorganization]
```

## Example

**Task**: "Organize the project notes by quarter"

**Output**: Files moved into Q1, Q2, Q3, Q4 folders with a report of changes
