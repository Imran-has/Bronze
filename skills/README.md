# Skills Directory

This folder contains skill templates that define how to handle different task types.

## Available Skills

| Skill | File | Purpose |
|-------|------|---------|
| Writing | `writing.md` | Create written content |
| Summary | `summary.md` | Summarize documents |
| Research Notes | `research-notes.md` | Compile research notes |
| Organization | `organization.md` | Organize and structure files |

## Adding New Skills

Create a new `.md` file with:

1. Skill name and purpose
2. When to use this skill
3. Step-by-step instructions
4. Output format
5. Example

## Using Skills

When processing a task:
1. Identify which skill applies
2. Read the skill file
3. Follow the instructions
4. Produce output in specified format
