# Skill: Research Notes

## Purpose

Compile and organize research notes from multiple sources.

## When to Use

- Task involves gathering information
- Task mentions "research", "compile", "gather", "collect"
- Need to organize information from multiple files

## Instructions

1. **Identify sources**
   - List all files/documents to review
   - Note the research topic or question

2. **Read and extract**
   - Read each source
   - Extract relevant information
   - Note source for each piece of info

3. **Organize by theme**
   - Group related information
   - Identify patterns or themes
   - Note contradictions or gaps

4. **Compile notes**
   - Create structured document
   - Include citations/sources
   - Highlight key findings

5. **Add synthesis**
   - Summarize overall findings
   - Note areas needing more research

## Output Format

```markdown
# Research Notes: [Topic]

**Date**: [date]
**Sources reviewed**: [count]

## Sources

1. [Source 1]
2. [Source 2]

## Findings

### Theme 1
- Finding (Source 1)
- Finding (Source 2)

### Theme 2
- Finding (Source 1)

## Synthesis

[Overall summary of findings]

## Gaps / Further Research

- [Area needing more research]
```

## Example

**Task**: "Compile research notes on competitor pricing"

**Output**: Organized notes with pricing data grouped by competitor, with sources cited
